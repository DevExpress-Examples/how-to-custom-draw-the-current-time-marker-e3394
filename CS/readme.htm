How to custom-draw the Current Time Marker

This example illustrates how to draw the Current Time Marker in the Time Ruler (http://documentation.devexpress.com/#WindowsForms/CustomDocument1737). To accomplish this task, handle the SchedulerControl.CustomDrawDayViewTimeRuler Event (http://documentation.devexpress.com/#WindowsForms/DevExpressXtraSchedulerSchedulerControl_CustomDrawDayViewTimeRulertopic) (calculate the Current Time Marker position and draw it manually in this event handler). Also, set the TimeRuler.ShowCurrentTime Property (http://documentation.devexpress.com/#CoreLibraries/DevExpressXtraSchedulerTimeRuler_ShowCurrentTimetopic) to false to suppress drawing of a standard Current Time Marker. See also: http://www.devexpress.com/scid=E2469
